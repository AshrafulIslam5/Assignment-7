Difference between local Storage and Session Storage

Ans: As far as I know, Local storage and Session storage are almost similar. Local storage allows us to keep important information that we need even after closing the window. Session storage is mainly used for keeping the data for the ongoing session only, if the window or the browser is closed, then the data stored in session storage will be lost. So, they have that one Difference and it is that Local storage keeps the data even after the window is closed, and Session storage keeps the data untill the window or the browser is closed.
